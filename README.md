# HR精灵 — AI 人力资源分析诊断助手

> 7 位 AI Agent 组成的人力资源分析团队，独立桌面应用。无需 EasyClaw，双击即用。

---

## 🎯 这是什么

一个独立的桌面软件。上传 HR 数据 → 7 个 AI Agent 接力分析 → 输出完整人力资源健康度诊断报告。

**完全独立运行**，不依赖 EasyClaw，只依赖 DeepSeek API。

---

## 📥 安装 & 运行

### 方式一：直接下载

从 [Releases](../../releases) 下载安装包，双击安装。

### 方式二：源码运行

```bash
# 1. 安装依赖
npm install

# 2. 启动
npm start
```

---

## 🔑 配置 API Key

1. 打开软件
2. 左侧面板 → 填入 DeepSeek API Key
3. 获取 Key：https://platform.deepseek.com/api_keys

一个 Key 即可驱动全部 7 个 AI Agent。

## ⚙️ 三种分析模式

| 模式 | 流程 | 适用场景 |
|------|------|---------|
| 🔍 快速概览 | 招聘专家 → 风险审计师 | 快速了解 HR 健康状态，2-3分钟 |
| 📊 标准分析 | 招聘专家 → 绩效分析师 → 薪酬顾问 → 报告总监 | 常规人力资源诊断报告 |
| 🏥 深度诊断 | 全套 7 Agent 接力 | 全面深度 HR 健康度诊断 |

## 👥 团队角色

| 角色 | 职责 |
|------|------|
| 🔍 招聘效能专家 | 分析招聘漏斗、渠道ROI、关键岗位储备 |

| 🔴 人力风险审计师 | 合规、用工、薪酬、组织风险全面排查 |
| 📋 HR报告总监 | 整合所有分析，出具终审诊断报告 |

## 📖 输入格式

将 HR 数据整理为纯文本（.txt 或 .md），建议包含：
- **员工概况**：总人数、学历分布、司龄分布
- **招聘数据**：招聘量、漏斗转化、渠道成本
- **离职数据**：离职率、关键岗位流失
- **绩效数据**：绩效等级分布、部门健康度
- **薪酬数据**：薪酬带宽、市场竞争力、人力成本率
- **培训数据**：培训投入、覆盖率、满意度
- **文化数据**：满意度、敬业度、价值观认同度
- **合规数据**：合同签署、社保合规、劳动纠纷

示例格式见 `demo-hr.txt`。
